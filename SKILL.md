# 36氪风格写作 Skill

## 账号画像

**36氪** — 新经济内容服务平台
- 核心定位：服务中国新经济参与者
- 受众：创业者、投资人、新经济从业者、互联网人
- 调性：**新锐 + 快 + 信息密度高 + 趋势敏感**
- Slogan：「让一部分人先看到未来」

---

## 核心写作特征

### 1. 标题公式

```
【趋势先知型】
→ "这件事，正在悄悄改变[行业/用户习惯]"
→ "[新技术/新玩法]，开始挑战[既有格局]"
→ "下一个[大机会]，藏在[意想不到的地方]"

【创投叙事型】
→ "[公司名]获[金额]融资，投资人为什么选它"
→ "一家[早期公司]，已经跑出了[惊人数字]"
→ "[赛道]新王：[公司名]凭什么"

【速度感型】
→ "只用[时间]，[公司]做到了[成就]"
→ "[数字]天，[产品]从0到[成就]"
```

### 2. 文章结构

```
【导语：用最快速度交代核心信息】
30-50字交代"这是什么、为什么重要、你需要知道"
节奏要快，让读者10秒内判断要不要继续读

【正文分段：多用小标题，少用长段落】
推荐分3-4段，每段一个核心信息点：

段落①：「这件事的全貌」
— 事件/公司/产品的核心信息摘要
— 关键数字置于醒目位置（首句/加粗）

段落②：「为什么现在，为什么重要」
— 市场时机/窗口分析
— 同赛道对比：别人在哪，它在哪

段落③：「增量信息：你在其他地方看不到的洞察」
— 独家采访/数据/内部消息
— 或：更深一层的商业逻辑拆解

段落④：「前景与风险」
— 不只是看好，要有风险提示
— 行业观察者/投资人的不同声音

【结尾：行动导向】
— 给读者一个能带走的「行动或判断」
— 或留下值得持续关注的一个观察维度
```

### 3. 写作风格原则

**必须做到**：
- 开头段必须有具体数字（融资额/用户量/增长率/估值）
- 每篇文章至少有一条「增量信息」（别处看不到的洞察）
- 行文节奏要快，不铺垫，直接说重点
- 涉及创业公司必须点出核心竞争差异（而非功能列表）

**绝对不能**：
- 大段铺垫行业背景（读者都是内行）
- 只写公司官方版本的"好消息"
- 不给风险评估
- 标题承诺的信息在正文里兑现不了

## STEP 流程

### STEP 1：确认参数
```
请告诉我：
1. 报道对象：公司/产品/赛道/融资事件？
2. 这篇文章的「增量信息点」是什么？（读者在别处看不到的）
3. 写给谁看：创业者 / 投资人 / 新经济爱好者？
4. 字数：快讯（800字）/ 深度（2000字）/ 深度长文（3000字）
```

### STEP 2：确立「增量信息」
在动笔前，明确：
- 这篇文章能给读者提供什么他们不知道的东西？
- 是独家数据？内部视角？深层商业逻辑分析？

### STEP 3：信息采集
重点渠道：
- 企业官方公告/招股书/投资人信息
- 36氪/虎嗅/晚点等竞品同题报道（做差异化）
- 行业数据：IT桔子、企查查、天眼查
- Twitter/LinkedIn/海外媒体（出海公司）

### STEP 4：按骨架写正文
遵循36氪节奏快、信息密度高的写作规范

### STEP 5：自检清单
```
□ 导语是否在50字内交代最重要信息？
□ 文章是否有至少一条"别处看不到"的增量信息？
□ 关键数字是否在显眼位置（加粗/段落首句）？
□ 是否包含了风险视角，不只写好消息？
□ 节奏是否足够快，有没有大段废话？
```
