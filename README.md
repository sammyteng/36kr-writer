# 📈 36氪（36kr）Writer Skill

> 让 AI 以36氪「新锐新经济内容」风格写商业创投深度报道的可复用 Skill。

## 简介

本 Skill 基于对36氪数百篇创投/科技/新经济报道的结构化解析，将36氪「快+深+信息密度高」的写作风格标准化为可复用 AI 写作指令，帮助快速生成符合新经济媒体调性的深度内容。

## 文件结构

```
36kr-writer/
├── SKILL.md                        # 主技能文件（核心写作哲学 + 流程）
└── resources/
    ├── playbook_funding.md         # 骨架一：融资事件报道
    ├── playbook_startup.md         # 骨架二：创业公司深度
    ├── playbook_industry.md        # 骨架三：赛道格局分析
    ├── playbook_product.md         # 骨架四：产品深度拆解
    └── playbook_trend.md           # 骨架五：新经济趋势洞察
```

## 36氪核心写作风格

| 维度 | 特征 |
|------|------|
| 标题 | 数字/趋势/发现感，吸引新经济从业者 |
| 节奏 | 快，30-50字导语交代最重要信息 |
| 信息密度 | 高，关键数字置于显眼位置 |
| 禁忌 | 大段铺垫、无风险视角、漏掉增量信息 |

## 核心写作哲学

- **增量信息优先**：每篇文章必须有读者在其他地方看不到的内容
- **速度感**：开头30秒内交代完最重要信息
- **数字驱动**：融资额/增速/市占率等数字在显眼处（首句/加粗）
- **风险平衡**：不只写好消息，必须包含风险视角

## 如何使用

```
使用 36kr-writer，帮我用36氪风格写一篇关于 [主题] 的创投/科技报道
```
